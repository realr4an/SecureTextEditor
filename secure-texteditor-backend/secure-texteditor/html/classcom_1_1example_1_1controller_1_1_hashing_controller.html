Detailed Description

REST-Controller zur Berechnung und Überprüfung von Hash-Werten. 

   Dieser Controller stellt API-Endpunkte zum Hashen von Texten und zur
   Überprüfung von Hash-Werten bereit.

◆ hash()

Map< String, String > com.example.controller.HashingController.hash ( @RequestBody Map< String, String > request)

Erstellt einen Hash-Wert für den gegebenen Text.

Parameters
requestEin JSON-Objekt mit den folgenden Parametern:
  • "text": Der zu hashende Text.
  • "macAlgorithm": Der Hash-Algorithmus (z. B. "SHA-256", "HMAC-SHA256", "AES-CMAC").
Returns
Ein JSON-Objekt mit:
  • "hashedText": Der berechnete Hash-Wert (Base64-kodiert).
  • "error": Falls ein Fehler auftritt, eine Fehlermeldung.

◆ verifyHash()

Map< String, Boolean > com.example.controller.HashingController.verifyHash ( @RequestBody Map< String, String > request)

Überprüft, ob ein gegebener Hash-Wert mit dem berechneten Hash übereinstimmt.

Parameters
requestEin JSON-Objekt mit den folgenden Parametern:
  • "text": Der ursprüngliche Klartext.
  • "macAlgorithm": Der verwendete Hash-Algorithmus.
  • "hash": Der erwartete Hash-Wert (Base64-kodiert).
Returns
Ein JSON-Objekt mit:
  • "isValid": true, wenn der Hash-Wert gültig ist, sonst false.
